Unix Shell

A Unix shell is a command-line interpreter, or CLI, that is used for managing files and automating some basic tasks.

The CLI I am using is called Git Bash.

Below are some examples of how it can be used...

How to list the files in a directory:

Use the command 'ls' to list files, as shown below, and press enter...

You can also use the command 'ls -F' followed by the file path, to show more specific information about the directory and files you want to list...

How to show and change the working directory:

Use the command 'pwd to show the current working directory, which stands for Print Working Directory...

In order to change the the current working directory, use the command 'cd' ...

You can also add the filename/path or use the command 'cd ..' to go back up one level in the file structure too...
